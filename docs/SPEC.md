# OpsCopilot - IT运维智能体产品需求文档

## 一、产品定位

**产品名称**：OpsCopilot - 运维智能体

**核心定位**：基于AI的智能运维助手，通过MCP工具调用运维系统API，实现自然语言交互、数据可视化和大屏展示。

**版本规划**：

| 版本 | 功能 | 状态 |
|------|------|------|
| MVP | 基础对话 + MCP工具调用 + 数据图表 | 开发中 |
| V1.0 | 大屏展示 + 预测分析 | 规划中 |
| V2.0 | Agent自治能力 | 规划中 |

---

## 二、核心功能

### 2.1 智能对话
- 自然语言理解用户意图
- 多轮对话上下文管理
- 意图识别：告警、资产、工单、知识库、统计

### 2.2 MCP工具系统
MCP (Model Context Protocol) 工具层，连接AI与运维系统：

| 工具名称 | 功能 | 数据来源 |
|---------|------|----------|
| `alert_query` | 查询告警列表 | 监控系统API |
| `alert_detail` | 告警详情分析 | 监控系统API |
| `asset_search` | 搜索资产 | CMDB API |
| `asset_topology` | 资产关联拓扑 | CMDB API |
| `ticket_create` | 创建工单 | 工单系统API |
| `ticket_query` | 查询工单状态 | 工单系统API |
| `knowledge_search` | 知识库搜索 | 知识库API |
| `stats_alert` | 告警统计 | 监控系统API |
| `stats_asset` | 资产统计 | CMDB API |
| `stats_ticket` | 工单统计 | 工单系统API |

### 2.3 数据可视化
- **对话内图表**：折线图、柱状图、饼图、表格
- **实时数据卡片**：关键指标展示
- **趋势分析**：时间序列图表

### 2.4 运维大屏
- **全局总览**：整体运维态势
- **告警大屏**：实时告警监控
- **资产大屏**：资产分布与状态
- **工单大屏**：工单处理效率
- **业务大屏**：各业务系统健康度

## 四、MCP工具规范

### 4.1 工具定义

```json
{
  "name": "alert_query",
  "description": "查询告警列表，支持按级别、状态、时间范围筛选",
  "parameters": {
    "level": "CRITICAL|WARNING|INFO",
    "status": "PENDING|PROCESSING|RESOLVED",
    "hours": "查询最近几小时的数据"
  }
}
```

### 4.2 工具执行流程

1. 用户发送消息
2. LLM 识别需要调用的工具
3. MCP 执行工具获取数据
4. 数据返回给 LLM
5. LLM 生成带图表的回复
