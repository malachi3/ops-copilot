# IT运维智能体产品需求文档 v2.1

## 一、产品定位

**产品名称**：OpsCopilot - 运维智能体

**核心定位**：基于AI的智能运维助手，通过MCP工具调用运维系统API，实现自然语言交互、数据可视化和大屏展示。

## 三、页面结构

| 页面 | 路径 | 功能 |
|------|------|------|
| 智能对话 | / | AI对话入口，支持图表展示 |
| 告警管理 | /alerts | 告警列表、统计、详情 |
